# 考核统计申诉案件标记方案

## 目标
1. **统计列表**：申诉通过的案件在"按时处置"和"按时结案"数字后以括号标注，如 `5（1）`
2. **反查列表**：对申诉通过的案件显示"申诉通过"标签

## 一、统计列表：数字后加括号标注

### 修改文件
`frontend/admin-web/src/views/evaluation/index.vue`

### 方案
在 `metricColumns` 中为需要标注的指标添加 `appealProp` 字段，标识对应的申诉计数字段：

```js
const metricColumns = [
  // ... 其他不变 ...
  { prop: 'onTimeHandleCount', label: '按时处置', metricKey: 'onTimeHandle', appealProp: 'appealOverdueHandleCount' },
  { prop: 'overdueHandleCount', label: '超时处置', metricKey: 'overdueHandle' },
  // ...
  { prop: 'onTimeCloseCount', label: '按时结案', metricKey: 'onTimeClose', appealProp: 'appealOverdueCloseCount' },
  { prop: 'overdueCloseCount', label: '超时结案', metricKey: 'overdueClose' },
  // ...
]
```

修改 `<template #default="{ row }">` 中的渲染逻辑，增加括号显示：

```html
<template #default="{ row }">
  <span v-if="canDrillCell(row, col.prop)" class="drill-link" @click="openDrill(row, col.metricKey, col.label)">
    {{ cellCount(row, col.prop) }}<template v-if="col.appealProp && cellCount(row, col.appealProp) > 0">（{{ cellCount(row, col.appealProp) }}）</template>
  </span>
  <span v-else>
    {{ cellCount(row, col.prop) }}<template v-if="col.appealProp && cellCount(row, col.appealProp) > 0">（{{ cellCount(row, col.appealProp) }}）</template>
  </span>
</template>
```

### 后端补充字段
当前 `CaseDeptStatisticsRow` 有 `appealOnTimeCloseCount` 和 `appealOverdueCloseCount`，但**缺少 handle 层面的申诉计数**。

需要在 `CaseReportServiceImpl.java` 的 SQL 中增加：
```sql
-- 申诉后超时处置（申诉通过，原超时，现按未超时统计）
SUM(CASE WHEN c.handle_finish_time IS NOT NULL AND c.deadline_time IS NOT NULL 
         AND c.handle_finish_time > c.deadline_time AND c.handle_timeout_exempt = 1 
    THEN 1 ELSE 0 END) AS appeal_overdue_handle_count,
```

在 `CaseDeptStatisticsRow.java` 中增加：
```java
private long appealOverdueHandleCount;
```

## 二、反查列表：案件行显示"申诉通过"标签

### 修改文件
`frontend/admin-web/src/views/evaluation/index.vue`

### 方案
在反查表格中增加一列"申诉"，当 `handleTimeoutExempt === 1` 时显示绿色 tag：

```html
<el-table-column label="申诉" width="100" align="center">
  <template #default="{ row }">
    <el-tag v-if="row.handleTimeoutExempt === 1" type="success" size="small">申诉通过</el-tag>
  </template>
</el-table-column>
```

后端 `CaseInfo` 实体已有 `handleTimeoutExempt` 字段，drill 接口返回时会自动包含，无需后端改动。

1. 重启后端（mvn install + 重启）

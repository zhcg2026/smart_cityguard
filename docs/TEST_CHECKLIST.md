## 排障速查

| 现象 | 处理 |
|------|------|
| 地图 `AMap is not defined` / `ERR_PROXY_CONNECTION_FAILED` | 关代理或 `*.amap.com` 直连；刷新页面 |
| 新 case 接口 404 | `mvn clean install -pl smart-cityguard-case -am -DskipTests` 后重启 8080 |
| DEPT 无批转按钮 | 状态须 `handle_finish`；`department_id` = `handle_dept_id` |
| 流程无接收人（旧数据） | 仅新操作有接收人；用新案复测 A 链 |
| 延期部门审无入口 | 确认已执行 `patch_case_adjustment_dept_review.sql` |

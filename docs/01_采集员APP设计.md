# 采集员APP模块设计

## 功能模块

### 一、今日提示
- 指挥中心发布的采集提示信息
- 实时查看

### 二、公文通告
- 指挥中心发布的通知公告
- 实时查看

### 三、问题上报（核心功能）

**上报字段：**
| 字段 | 说明 |
|------|------|
| 问题类型 | 选择立结案标准（大类 → 小类） |
| 立案条件 | 选择具体立案条件（同一小类可能有多个） |
| 定位 | GPS自动定位 + 可手动拖动调整 |
| 问题描述 | 采集员填写 |
| 附件 | 拍照/录像 或 从手机选择（单个10MB，总量50MB） |

### 四、我的任务

**任务类型：**

| 类型 | 场景 | 反馈内容 | 结果处理 |
|------|------|----------|----------|
| 核查任务 | 渠道案件无照片，需确认问题是否存在 | 照片+说明+是否存在 | 存在→立案 / 不存在→作废 |
| 核实任务 | 已处置案件，需确认处置是否到位 | 照片+说明+是否到位 | 到位→结案 / 不到位→回退 |

**任务列表分类：**
- 未完成
- 已完成
